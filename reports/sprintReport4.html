Narrative of Accomplishments and Challenges

In this sprint, we completed the final remaining project requirements and completed product testing. The greatest challenge was setting up the A/B test. We mainly had difficulties in setting up the test with Google Analytics/Google Optimize, but were ultimately able to troubleshoot these issues and run the test. We tracked page views for the donations page (the destination of the Donate/Support Us link) as a proxy for link clicks, after confirming with Professor Jensen that the bots would not navigate to the donations page directly, so the page views would be an appropriate proxy.
